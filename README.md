# Wizard

function for msgs

## Getting Started

add jitpack.io in allprojects, in project gradle

    allprojects {
        repositories {
            google()
            jcenter()
            maven { url 'https://jitpack.io' }
        }
    }

  

### Installing

 implement lib in yout module gradle
 
     implementation 'com.github.quanticheart:Wizard:1.0.11'
